# nestx server
Server side for nestx
### Quick start

```bash

# install the repo with npm
$ npm install

# start the server
$ npm start

```

### Other commands

```bash

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

### Test

```
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
